Concept A: Color Palette

Primary Color #3E5C4A 信頼感と自然を感じさせる深い緑。予約エリアの背景に使用。
Accent Color #E67E22 ボタンや仕切り線に使用。フレッシュな活力を与えます。
Background Color #FDFDFB 清潔感のある暖かいクリーム。広がりと落ち着きを演出。
Overlay Color rgba(62, 92, 74, 0.55) ヒーロー画像の透かしに使用。写真にブランドの統一感を与えます。
Design Strategy:
「ボタニカル＆フレッシュ」を掲げ、予約エリアにブランドの象徴である深緑を大胆に配置。信頼できる「プロの技術」を強調した構成です。

Concept B: Color Palette

Primary Color #5D4037 温かみと安心感のある深いブラウン。予約エリアの背景に使用。
Accent Color #D35400 テラコッタ調のオレンジ。深みのあるアクセントとして機能。
Background Color #F5F1E9 落ち着いたサンドベージュ。リラックス効果を高めます。
Overlay Color rgba(93, 64, 55, 0.65) ブラウン調の透かし。写真にアンティークで上質な雰囲気を与えます。
Design Strategy:
「アース＆ウォーム」をテーマに、予約エリアを落ち着いた茶色で構成。ゆったりとしたプライベートな「自分時間」を大切にする層へ向けたデザインです。
