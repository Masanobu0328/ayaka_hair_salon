## 📐 プロジェクト概要

### コンセプト
ゴールドコースト・サウスポートで営業する日本人美容師のプライベートサロン。
「日本の繊細な技術」×「オーストラリアの開放感」を融合した、上品かつ温かみのあるWebサイト。

### ゴール
- 第一印象で信頼と高級感を伝える
- メニュー・価格が一目でわかる
- Square経由でオンライン予約が完結する
- 日本語/英語のバイリンガル対応
- モバイルファーストで完全レスポンシブ

### 技術スタック
| 領域 | 技術 | 理由 |
|------|------|------|
| **フレームワーク** | HTML + CSS + Vanilla JS | シンプル・高速・保守しやすい |
| **予約** | Square Appointments（iframe埋め込み） | API鍵不要・セキュリティ安全 |
| **地図** | Google Maps（iframe埋め込み） | 取得済み |
| **多言語** | Client-side i18n（data-i18n属性） | サーバー不要・静的サイト対応 |
| **ホスティング** | Vercel or Netlify | 無料・CDN・カスタムドメイン対応 |
| **フォント** | Google Fonts（Playfair Display, Inter, Noto Sans JP） | ブランドイメージに合致 |

### 参照スキル
- `service_business_lp` → セクション構成・デザインシステム
- `i18n_bilingual` → 日英切り替え
- `contact_form` → 問い合わせフォーム（必要に応じて）

## 🎨 デザイン仕様

### カラーパレット
```
Primary    : #3E5C4A  (Salon Green)    → 信頼・自然・安定
Background : #FDFDFB  (Warm Cream)     → 清潔・目に優しい
Text       : #2C3E50  (Deep Slate)     → 可読性・高級感
Accent     : #D4AF37  (Soft Gold)      → 上品なアクセント
White      : #FFFFFF  (Pure White)     → カード背景
```

### タイポグラフィ
```
Headings : Playfair Display (セリフ体) → エレガンス・専門性
Body     : Inter (英語) / Noto Sans JP (日本語) → モダン・可読性
```

### デザイン原則
1. **余白の贅沢さ** — 詰め込みすぎない。GCの開放感を表現
2. **ミニマリズム** — ロゴなし戦略。フォントと色でブランドを確立
3. **写真中心** — サロンの技術力は写真で証明する
4. **マイクロアニメーション** — スクロール時のフェードイン・ホバーエフェクト
5. **モバイルファースト** — 来客の80%以上がスマホ想定
